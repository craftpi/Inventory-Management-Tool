Dieses interne Verwaltungstool nutzt spezialisierte technische Dienstleister, um die Funktionalität zu gewährleisten:

1. Supabase (Datenbank & Login)
Wir nutzen Supabase (Supabase, Inc., USA) zur Speicherung der Bestandsdaten und für die Benutzeranmeldung. Hierbei wird technisch bedingt Ihre IP-Adresse übertragen und ein Authentifizierungs-Token im lokalen Speicher Ihres Browsers abgelegt.
Details: Supabase Privacy Policy

2. GitHub (Hosting)
Die Bereitstellung der Website-Dateien (HTML/JS) erfolgt über GitHub Pages (GitHub, Inc., USA). GitHub kann Protokolldaten über Zugriffe (IP-Adresse, Browser-Typ) erheben, um den Dienst sicher und zuverlässig bereitzustellen.
Details: GitHub Privacy Statement
